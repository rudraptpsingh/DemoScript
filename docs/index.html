How it works

Three steps. Real video.

1

Discover elements

Run demoscript capture <url> to scan any page and get a list of CSS selectors you can target.

2

Write your script

Define steps in a JSON file: scroll here, zoom there, highlight this. Set durations and easing curves.

3

Render

Run demoscript render --script demo.json and get an MP4 or GIF. Headless Chromium + FFmpeg does the rest.

Actions

🔍

zoom-in / out

✨

highlight

↔

pan

Smooth viewport panning

🖱

cursor-move

Animate cursor to an element

👆

click

Move to element and click it

⏸

wait

Hold the frame with optional text

💬

annotations

Text overlays on any step

Use cases

Replace manual recordings

Product demos

Ship a landing page update and regenerate the demo video in CI. Always up to date, zero manual effort.

Docs and changelogs

Embed GIFs in your README or changelog that show exactly what changed. Script once, render on every release.

Sales and marketing

Generate personalized demo videos per prospect by swapping the URL. Same script, different sites.
